# K007 中国二级票务平台概览 (2026版)

## 一、 核心比价平台 (重点开发对象)
这些平台具有结构化的数据，是我们“比价分子技能”的首选目标。

| 平台名称 | 业务模式 | 抓取难度 | 核心价值 |
| :--- | :--- | :--- | :--- |
| **摩天轮票务** | 平台撮合/加价/折价 | 中 (有Web/H5) | 票源最稳，价格变动最快，是行情风向标。 |
| **票牛** | 卖家入驻/平台担保 | 中 | 与摩天轮形成互补，常有不同力度的优惠。 |
| **优票 (Youpiao)** | 垂直二手票务 | 高 | 在特定品类（如话剧）中有独特票源。 |
| **票星球 (一级/分销)** | 官方分销/二级流转 | 高 | 经常有热门项目的独家/优先购票权。 |

## 三、 业内调票系统与 B2B 平台 (专业从业者阵地)
这些系统反映了票务代理商内部的真实“底价”和“库存”。

| 平台名称 | 核心角色 | 自动化价值 |
| :--- | :--- | :--- |
| **票务管家** | 行业分销/库存同步 | 监控其分销接口可获取最真实的市场供给。 |
| **票行天下** | 老牌调票系统 | 传统票务公司之间交换库存的中枢。 |
| **大河票务** | 大型票源商 | 获取大批量票务行情的重要参考。 |

## 四、 银行/企业内购与隐秘渠道
往往被大众忽略，但票源非常稳健。

| 渠道类型 | 代表对象 | 特点 |
| :--- | :--- | :--- |
| **银行金融 App** | 招商银行/工商银行 | 专属购票专区，常有溢价项目的原价额度。 |
| **大厂福利平台** | 京东/阿里福利商城 | 内部员工优先，是市场上散票的重要来源。 |

## 五、 社交/C2C 与跨境流转
这些平台数据非结构化，主要用于“找客分子技能”和“行情监控”。

| 平台名称 | 交易特性 | 自动化方向 |
| :--- | :--- | :--- |
| **闲鱼 (Xianyu)** | 纯C2C，信息极度碎片化 | 关键词监控，识别真实卖家与竞对。 |
| **小红书 (XHS)** | 内容驱动，信任度高 | 评论区截流，私信触达，品牌展示。 |
| **微博 (Weibo)** | 粉丝聚集地，实时性极强 | 超话关键词抓取，实时行情判断。 |
| **Viagogo/StubHub** | 国际二级平台 | 专门针对港澳/海外场次的溢价监控。 |

## 三、 比价开发建议 (Inventory Workflow)
1. **第一优先级**：摩天轮 & 票牛。这两个平台的 H5 接口相对容易适配，可以快速实现“同场次、同档位”的自动比价。
2. **策略**：
   - 抓取**大麦**作为原价基准线。
   - 抓取**摩天轮/票牛**计算溢价率/折扣率。
   - 实时输出“比价报告”存入项目的 `inventory/` 目录。
